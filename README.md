# AI Resume Analyzer

Doğal Dil İşleme (NLP) ve Makine Öğrenmesi kullanılarak geliştirilen akıllı CV analiz ve ATS değerlendirme sistemi.

Bu proje, CV ile iş ilanı arasındaki uyumu analiz ederek kullanıcıya çeşitli geri bildirimler sunmaktadır.

---

# Proje Özellikleri

✅ CV ve iş ilanı eşleştirme sistemi

✅ TF-IDF tabanlı benzerlik analizi

✅ Weak CV (zayıf CV) tespiti

✅ Eksik teknik beceri analizi

✅ NLP tabanlı metin ön işleme

✅ ATS mantığına uygun temel değerlendirme sistemi

---

# Kullanılan Teknolojiler

- Python
- NLP (Natural Language Processing)
- Scikit-learn
- TF-IDF Vectorization
- Cosine Similarity
- Gradio

---

# Projenin Çalışma Mantığı

1. Kullanıcı CV metnini girer
2. Kullanıcı iş ilanı açıklamasını girer
3. Sistem:
   - Benzerlik skorunu hesaplar
   - Eksik becerileri tespit eder
   - Zayıf ifadeleri analiz eder
